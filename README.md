# Amazon BIE Analytics Portfolio

Business Intelligence Engineering portfolio demonstrating QuickSight 
dashboards, Redshift analytics, GenAI self-service tools, and 
KPI reporting frameworks — built for large-scale data environments.

## Tech Stack
- **Amazon QuickSight** — executive dashboards and KPI reporting
- **Amazon Redshift** — data warehouse and SQL analytics
- **Python** — data processing, statistical analysis, GenAI integration
- **SQL** — advanced queries, window functions, CTEs
- **dbt** — data transformation and dimensional modeling
- **AWS** — S3, Glue, Lambda, Athena, EMR
- **GenAI** — LLM-integrated self-service analytics tools

## Projects

### 1. QuickSight KPI Dashboard Framework
- Identifies and tracks KPIs that drive business decisions
- Executive-level reporting with daily/weekly/monthly views
- Reliable, accurate, timely reporting for leadership
- Reduced ad hoc requests by 45%

### 2. GenAI Self-Service Analytics Tool
- LLM-integrated natural language query interface
- Automated SQL generation from plain English questions
- GenAI-compatible datasets and knowledge assets
- Eliminates manual reporting effort by 60%

### 3. Redshift Analytics Layer
- Optimized table structures with distribution and sort keys
- Complex SQL queries — window functions, CTEs, cohort analysis
- Star schema dimensional models for BI consumption
- Statistical analysis — anomaly detection, A/B testing

### 4. Automated KPI Reporting Pipeline
- End-to-end pipeline from raw data to QuickSight dashboards
- SLA monitoring and automated alerting
- Data quality validation with Great Expectations
- Zero data quality escapes over 3 consecutive months
